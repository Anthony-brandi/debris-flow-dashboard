# Debris Flow Dashboard

A real-time post-fire debris flow risk assessment tool integrating Google Earth Engine satellite analysis, empirical volume modeling, and live geospatial visualization.

> **Current Version:** v1.0.0

---

## Scientific Background

This tool applies the **Gartner et al. (2008)** empirical model to estimate post-fire debris flow volume:

    ln(V) = 4.22 + 0.39*sqrt(i15) + 0.36*ln(A) + 0.13*sqrt(B*R)

| Variable | Definition |
|---|---|
| V | Predicted debris flow volume (m3) |
| i15 | Peak 15-min rainfall intensity (mm/h) |
| A | Watershed area (km2) |
| B | Fraction of basin burned at moderate-high severity |
| R | Local relief — max minus min elevation (m) |

**Reference:** Gartner, J.E., Cannon, S.H., Santi, P.M., & Dewolfe, V.G. (2008). Empirical models to predict the volumes of debris flows generated by recently burned basins in the western U.S. *Geomorphology*, 96(3-4), 121-139.

## Data Sources

| Dataset | Catalog | Resolution |
|---|---|---|
| Digital Elevation Model | USGS SRTM (`USGS/SRTMGL1_003`) | 30 m |
| Burn Severity (dNBR) | ESA Sentinel-2 SR (`COPERNICUS/S2_SR_HARMONIZED`) | 10 m |
| Soil Erodibility | OpenLandMap (`OpenLandMap/SOL/SOL_TEXTURE-CLASS_USDA-TT_M/v02`) | 250 m |
| Debris Flow Inventory | USGS Post-Fire Debris Flow Database | Point observations |

---

## App Modules

| Module | Description |
|---|---|
| 1. Incident Briefing | Fire metadata, perimeter map, mission parameters |
| 2. Spatial Modeling Lab | Live GEE satellite imagery, slope and dNBR analysis |
| 3. Predictive Debris Flow Modeling | Gartner volume model with watershed inputs |
| 4. Model Validation | Predicted vs. USGS observed comparison, R2, RMSE, NSE |
